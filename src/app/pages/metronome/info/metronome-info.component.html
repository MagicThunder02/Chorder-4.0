Metonome Info What is Metronome? Metonome is a free Metronome and polyrhythm generator. How to use it? TEMPO All the bpm options 120 Bpm Here you can check the current bpm value. You can change its value using the slider, the buttons or the tap feature {{'TAP'}} Tap Tap along with a song beat to adjust the metronome to the song bpm TRAIN Train yourself increasing the bpm over time INCREASE Activates the train mode 160 Final Sets when the metronome will stop increasing 10 Step Sets the number of bpm added each time 4 Measures Sets the number of measure between each increment CUSTOMIZE Change the color and sound of every metronome 4 Beats Changes the number of beats for each measure Color Changes the color of this metronome Sound Changes the sound of this metronome
